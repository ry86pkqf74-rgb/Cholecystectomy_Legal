# Cholecystectomy_Legal

Westlaw full-text legal case datasets staged for LLM extraction on cholecystectomy and gallbladder/bile duct injury matters.

## Contents

### `Cholecystectomy/` (12 files)
Westlaw "full text items for cholecystectomy" result sets. Filenames preserved as exported (e.g. `Westlaw - 100 full text items for cholecystectomy.rtf`, `...cholecystectomy(1).rtf`, `...cholecystectomy2.rtf` ... `...cholecystectomy11.rtf`, `Westlaw - 95 full text items for cholecystectomy12.rtf`).

### `Gallbladder_Bile_Duct/` (4 files)
- `Westlaw - 100 full text items for gallbladder bile duct.rtf`
- `Westlaw - 100 full text items for gallbladder bile duct(3).rtf`
- `Westlaw - 33 full text items for gallbladder bile duct.rtf`
- `Westlaw - 49 full text items for gallbladder bile duct injury.rtf`

## Notes
